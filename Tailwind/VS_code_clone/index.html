VSCode is a versatile editor with great support for many programming languages and a vast extensions marketplace. Its UI and keyboard shortcuts can be customized, making it a powerful and flexible tool for any development work.

Meet IntelliSense.

Go beyond highlighting and autocomplete with InteliiSense, which provides smart completions based on variable types,function definitions, and imported modules.

Print statement debugging is a thing of the past.

Debug code right from the editor. Launch or attach to your running apps and debug with break points, call stqcks, and an interview console.

Git commands built-in.

Working with Git and other SCM providers has been easier.Review diffs, stage files, and make commits right from the editor. Push and pull from any hosted SCM service.

Extensible and customizable.

Want even more features? Install extensions to add new languages, themes, debuggers, and to connect to additional services. Extensions run in separate processes, ensuring they won't slow down your editor. Learn more about extensions.
